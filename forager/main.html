    Forager, a game by KudosInc (Guy)

    Forager is the third "completed" game I've made using Javascript. It also uses the popular JS library jQuery.
    I made the game to expand my Javascript skills after becoming adept at CSS and HTML.
    Some simple tips/a guide:
    1. Use the 'look around' button to forage for some supplies.
    2. Click these supplies to add them to your inventory. Once an item is in your inventory, different actions are unlocked as you can combine items, use them, or leave them be until you find another combination.
    3. As you forage, your herbology skill increases. Keep foraging and you gain the ability to find different types of items.
    4. There are currently 21 obtainable items.
    The story so far:
    You have crash landed at the top of a cliff, next to an dense forest. For some reason, this forest contains lots of plants that wouldn't usually be found near each other (so sue me). You are pretty bad at identifying things. For now, all you can really find are brightly coloured berries. Don't worry, you'll get better at that soon.



    Important Thing to Note:
    This is an unfinished game. Currently there is no prestige mechanic, althought it possible to "beat" the game. I have engineered the code in such a way that it would be faily easy to add new content -- but the game as it stands is just a proof of concept of the mechanics involved.
